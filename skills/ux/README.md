# ux — UX チーム

子タスク 1 件あたり `ux-delivery` workflow **v2**。PM がハブ。**Figma-first** でビジュアル品質を届ける。

| slug | 役割 |
|------|------|
| ux-pm | 進行・profile・**サブタスク分解**・委譲・artifact 公開・完了 |
| ux-designer | **Figma 画面 UI** · ux-spec companion（flagship は複数案） |
| design-system-owner | **Figma DS** · design-system.md · 任意 Code Connect |
| ux-reviewer | **design_quality** · ux_spec · ux_implementation（開発から委譲可） |

## delivery profile

| profile | 用途 |
|---------|------|
| `flagship` | 新規 Web · ブランド体験（複数案必須） |
| `standard` | 通常の画面追加・改修（既定） |
| `lite` | 既存 DS 流用 · spec 更新のみ |
