# data-analyst SKILL

**独立スキル:** analytics-pm から委譲された **探索・分析**。

## 責務

- 仮説検証・探索的分析
- BI ダッシュボード作成
- ステークホルダー向けインサイトの文書化

成果物: 分析ノート、BI ダッシュボード（別リポジトリまたは BI ツール）

完了後: **analysis-reviewer**（`review_kind: analysis_insights`）→ analytics-pm へ報告。

## Asana

`comment_task.py --agent data-analyst --skill skills/data-analyst/SKILL.md`

## 起動例

```
data-analyst: KPI に基づき探索分析とダッシュボードを作成し、analysis-reviewer へレビュー依頼してください。
```
