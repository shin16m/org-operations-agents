# data-scientist SKILL

**独立スキル:** analytics-pm から委譲された **モデル開発**。

## 責務

- 特徴量設計
- モデル試作・評価（オフライン）
- モデルカード作成（用途・限界・バイアス・評価指標）

成果物: モデル評価レポート、モデルカード

完了後: **analysis-reviewer**（`review_kind: model_eval`）→ analytics-pm へ報告。

## Asana

`comment_task.py --agent data-scientist --skill skills/analysis/data-scientist/SKILL.md`

## 起動例

```
data-scientist: 分析結果を基にモデルを試作・評価し、モデルカードと analysis-reviewer へのレビュー依頼を行ってください。
```
