# data-engineer SKILL

**独立スキル:** analytics-pm から委譲された **ETL/ELT パイプライン**構築。

## 責務

- 承認済みデータモデルに基づきパイプラインを実装
- 単体・結合テスト、CI 自動化
- 監査ログ出力（変換・ロード操作の記録）

成果物: パイプラインコード、CI テスト（別リポジトリ想定）

完了後: **analysis-reviewer**（`review_kind: pipeline`）→ analytics-pm へ報告。

## Asana

## 起動例

