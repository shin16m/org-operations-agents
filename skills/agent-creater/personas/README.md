# Personas（メタ用・共有）

**新規のペルソナファイルはエージェントごとのフォルダに置く:** `skills/agent-creater/agents/<agent-slug>/personas/`

このディレクトリは、メタスキル全体で共有する補足や、まだスラッグ化していないドラフト置き場として使ってよい。具体例（Asana Buddy）は `../agents/asana-buddy/personas/` を参照。

運用上の注意:
- ペルソナ内にAPIトークン等の機密情報を含めないでください。
- 実際のAsana操作を行う前に、必ずユーザー確認を行うフローにしてください。
