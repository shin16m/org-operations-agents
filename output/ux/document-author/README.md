# document-author — UX 文書テンプレート

`document-author` スキルが参照する文書構成の SSOT（UX 策定）。

**git 管理:** 新 clone でも参照可能。`.gitignore` で `output/ux/document-author/**` のみ例外許可。

| ファイル | 用途 |
|----------|------|
| [template-general-doc.md](template-general-doc.md) | レポート・説明文書（汎用） |
| [template-planning-doc.md](template-planning-doc.md) | 依頼者向け企画書（Epic 連動） |
| [template-system-catalog.md](template-system-catalog.md) | システムカタログ（skills / workflow / registry） |
| [diagram-guide.md](diagram-guide.md) | 図表（Mermaid・表）の使い分け |

**profile:** lite（Figma 不要 · 情報設計のみ）

**Canvas:** 定量データ・多段比較・インタラクティブ探索が主目的のときは Cursor Canvas を検討。通常の企画書・カタログは Markdown + Mermaid で十分。
